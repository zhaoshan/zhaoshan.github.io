新規口座開設＆「給与振込口座」指定で1,500円キャッシュプレゼントプログラム

例えば9月にスターワン口座の開設を申し込んだ場合…

9月末日までに口座開設の受付を完了し、口座開設完了後の翌々月（11月）までに、1回1万円以上の給与振込があれば1月に1,500円キャッシュプレゼント！

参加条件とご注意事項
対象者

2023年9月1日以降、新たにインターネットから「スターワン口座」を開設された方
条件 口座開設月の翌々月末までに1回1万円以上の給与を受け取られた方
プレゼント 口座開設月の4ヵ月後の月末までに、給与振込口座に1,500円をご入金
ご注意事項
  • インターネットからの口座開設申込のみ対象となります。
    ※郵送・店頭での口座開設は対象外
  • キャッシュプレゼントはスターワン円普通預金口座へ入金いたします。
  • 過去に本プログラムで1,500円プレゼントを受けたことがある方は対象外となります。
  • ”すでに口座をお持ちの方も「給与振込口座」指定で1,500円キャッシュプレゼント！”(2023年9月1日(金)～2023年12月31日(日)実施)に参加された方は本プログラムの対象外となります。
  • キャッシュプレゼント実施時に口座を解約されている方はキャッシュプレゼント対象外となります。
  • キャッシュプレゼントによる景品は課税の対象となる場合があります。詳しくは所轄税務署等にご相談ください。

お手続きは、かんたん2ステップ！

必要なお手続きは、スターワン口座の開設と、お勤め先へのご連絡のみ。東京スター銀行への申請は不要です。

ステップ1
スターワン口座を開設
最短3営業日で、お手元のスマートフォンからお手続きが可能です。既に口座をお持ちの方はステップ２へ。
ステップ2
お勤め先に給与振込口座変更を依頼
お勤め先からのお振り込みが「給与振込」の電文（以下に記載の通信種目コード）の場合、摘要に「給与振込（または「給振」）」と記載され、優遇金利の対象となります。摘要はお取引明細からご確認が可能です。
なお、摘要が「振込」の場合でも、備考欄1または2に「ｷﾕｳﾖ」もしくは「ｷｳﾖ」と記載があれば、優遇金利の対象となります。備考欄は銀行間の送金やり取りの際に使用する項目のため、お客さま、またはお勤め先において入力、指示、および内容の確認はできません。

よくあるお問い合わせ

Q
給与を全額振り込まないと優遇金利の対象にはなりませんか？
A

給与を全額ではなく一部のみお振り込みいただいた場合でも、優遇金利を適用いたします。たとえば、給与30万円のうち、5万円だけをスターワン口座に給与振込した場合も優遇金利の対象です。ただし、プログラムやキャンペーン等にご参加いただく場合、最低お振込金額などの各種条件がある場合がございます。詳しくは、キャンペーン・プログラム一覧をご確認ください。

Q
給与の受取口座をスターワン口座に変更する場合は、どうすればいいですか？
A

勤務先のご担当者さまへ給与振込口座の変更手続きについてご確認いただき、指定している口座を変更してください。
支店番号・口座番号などの口座情報は、キャッシュカードやインターネットバンキングからご確認いただけます。

Q
優遇金利を受けるために東京スター銀行に申請することはありますか？
A

ございません。スターワン円普通預金のお取引明細の摘要に「給与振込（または「給振」）」と記載があれば、優遇金利を適用いたします。ただし、摘要が「振込」の場合でも当行が給与振込と確認できた場合には優遇金利を適用いたします。

なお、通帳やお取引明細に記載される摘要は、各金融機関により異なります。当行は、お勤め先からのお振り込みが「給与振込」の電文の場合、摘要に「給与振込（または「給振」）」と記載されます。お振込手続きについては、お勤め先のご担当者さまへご確認ください。

Q
優遇金利はいつから適用されますか？
A

給与振込の確認ができたお客さまに対して、翌月、原則第5営業日より金利を優遇いたします。給与振込がない場合、翌月の原則第5営業日より通常金利となります。なお当行の都合により金利適用日を変更することがあります。
給与振込口座指定でキャッシュプレゼントの企画には、給与振込金額が1回1万円以上等の条件がありますのでホームページにてご確認ください。

Q
給与振込すれば、円普通預金に預けている資金すべてに優遇金利が適用されますか？
A

給与振込をしていただきますと、お振り込みをいただいたスターワン円普通預金口座の残高すべてに優遇金利を適用いたします。
ただし、スターワン円普通預金のお取引明細の摘要に「給与振込（または「給振」）」と記載があることが条件となります。摘要が「振込」の場合でも当行が給与振込と確認できた場合には優遇金利を適用いたします。
なお、通帳やお取引明細に記載される摘要は、各金融機関により異なります。当行は、お勤め先からのお振り込みが「給与振込」の電文の場合、摘要に「給与振込（または「給振」）」と記載されます。お振込手続きについては、お勤め先のご担当者さまへご確認ください。

Q
パート・アルバイトの給与でも優遇金利の対象になりますか？
A

はい、スターワン円普通預金のお取引明細の摘要に「給与振込（または「給振」）」と記載があれば、優遇金利を適用いたします。ただし、摘要が「振込」の場合でも当行が給与振込と確認できた場合には優遇金利を適用いたします。

優遇金利について

  • 変動金利となります。
  • 当行の都合により、予告なく中止、または内容を変更する場合があります。
  • スターワン円普通預金のお取引明細の摘要に「給与振込（または「給振」）」と記載があるお客さま、もしくは「振込」と記載があるお客さまのうち備考欄1または2に「ｷﾕｳﾖ」もしくは「ｷｳﾖ」と記載があるお客さまに対して、翌月、原則第5営業日より金利を優遇します。どちらも該当しない場合、翌月の原則第5営業日より通常金利となります。なお当行の都合により金利適用日を変更することがあります。
  • 通帳やお取引明細に記載される摘要は、各金融機関により異なります。当行は、お勤め先からのお振り込みが「給与振込」の電文の場合に、摘要に「給与振込（または「給振」）」と記載されます。お振込手続きについては、お勤め先のご担当者さまへご確認ください。
  • 税引後金利は税率20.315％（国税15.315％、地方税5％）で計算しています。なお国税のうち0.315％分は復興特別所得税の導入によるものです。税引後金利は表示の簡素化のため小数点以下第五位切り捨てにて表示しておりますので、実際の年利とは異なる場合があります。
  • 預け入れ金額が小額で、利息額が1円未満と計算された場合、利払いは行われません。
  • 本預金は預金保険の対象であり、当行にお預け入れいただいている他の預金保険の対象となる預金等と合算して、預金者さま一人あたり元本1,000万円までと保険事故発生日までの利息が保護されます。
  • 詳しくはホームページにご用意していますスターワン円普通預金商品説明書[PDF:78KB]をご覧ください。
