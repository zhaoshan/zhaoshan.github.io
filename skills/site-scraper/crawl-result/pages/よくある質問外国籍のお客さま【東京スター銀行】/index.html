口座開設

Q
口座開設について、いま住んでいる住所と本人確認書類の住所が違う場合、どちらの住所で入力したらいいのでしょうか？
A

口座開設のお申し込み時に、いま住んでいる住所と本人確認書類の住所が違う場合、口座開設を行うことはできません。
口座開設には本人確認書類記載の住所と現在お住まいになられている住所が同一であることが必要になります。

Q
口座の開設のお申し込みについて、キャッシュカードなど郵送物は申込書に記載した住所とは別の住所へ送っていただきたいのですが。
A

スターワン口座の開設のお申し込みについて、キャッシュカードなどの郵送物は、ご本人さま確認も兼ねておりますので、ご入力いただいた住所以外への発送はできません。

Q
口座開設を申し込みました。本人限定受取郵便は、本人以外の家族でも受取可能でしょうか？
A

本人限定受取郵便のお受け取りはご本人さまのみとなります。

Q
支店に行かなくても取り引きできますか？
A

支店に行かなくてもお取り引き可能です。
テレホンバンク、あるいはインターネットバンキングでお取り引きいただける商品もございます。
なお、インターネットバンキングをご利用いただくには、利用登録手続き（オンラインサインアップ）が必要です。
2019年8月16日（金）正午以降にインターネットから新規口座開設されたお客さまは、口座開設と同時にオンラインサインアップのお手続きが完了しています。
口座開設のお申込後、3~7営業日でご登録のメールアドレスにインターネットバンキングのログインIDをお送りいたします。こちらのメールをお受取後、すぐにインターネットバンキングをご利用いただけます。

  • ※審査状況により、７営業日以上お時間を要する場合もございます。
  • ※本人確認書類２点でインターネット口座開設の申し込みの場合は、キャッシュカードのお受け取りが確認できるまでインターネットバンキングでお振り込み、口座引き落としはご利用いただけません。
Q
スターワン口座について、キャッシュカードや暗証番号通知ハガキを支店で受け取ることはできますか？
A

いいえ。スターワン口座のキャッシュカードや暗証番号通知ハガキを当行の本支店ではお受け取りいただくことはできません。郵送のみでお受け取りいただけます。

外貨預金

Q
外貨預金を現金で引き出せますか？
A

外貨現金のお取り扱いはございません。
外貨預金を現金で引き出すことはできません。

Q
外貨預金は預金保険制度の対象ですか？
A

外貨預金は預金保険制度の対象外です。

Q
外貨預金は預金保険制度の対象外ですが、預け入れの金融機関が破綻したらどうなりますか？
A

円預金のうち、預金保険制度の対象とならない部分（元本1,000万円を超える部分）と同じ扱いとなり、破綻した金融機関の財産の状況に応じて支払われるため、一部カットされる可能性があります。
詳しくは預金保険機構のホームページ をご覧下さい。

Q
外貨預金の税金はどのようにかかるのですか？
A

外貨預金の税金のうち、利子所得は源泉分離課税20.315％（国税15.315％、地方税5％）として課税されます。
なお国税のうち0.315％分は復興特別所得税の導入によるものです。
為替差益は雑所得として、確定申告による総合課税の対象です。
ただし、年収2,000万円以下の給与所得者の方で為替差益を含めた給与所得・退職所得以外の所得が年間20万円以下の場合は申告不要です。
為替差損は、他の黒字の雑所得から控除できます。他の所得区分との損益通算はできません。
くわしくはお客さまご自身で公認会計士・税理士にご相談ください。

Q
外貨預金の引き出しはどのようにすればいいですか？
A

外貨預金のお引き出しは円資金への振り替えをしていただくか、同一通貨での送金もお受けできます。外貨現金でのお引き出しはできません。

Q
外貨の普通預金は、外貨のまま海外で引き出すことはできますか？
A

海外で当行にお預け入れいただいている外貨預金を引き出すことはできません。

Q
東京スター銀行で外貨預金を始めるにはどうすればいいですか？
A

東京スター銀行で外貨預金を始めるには、スターワン口座を開設いただき、お振り込みもしくはキャッシュカードでのご入金により、円資金を口座にご入金ください。

口座開設お申し込みはこちら

その後、ご来店、テレホンバンク、インターネットバンキング（東京スターダイレクト）でお手続きください。他の金融機関にお預け入れされている米ドル、オーストラリアドルやユーロ等の外貨預金も、外貨建てのまま当行へご送金できます。ただし、別途送金手数料がかかります。

Q
他行に持っている外貨預金からも東京スター銀行の外貨預金に預けられますか？
A

はい。他の金融機関でお持ちの米ドル、オーストラリアドルやユーロ等の外貨預金も当行へ外貨建てのままご送金いただき、お預け入れいただくことができます。

各種サービス・お手続き

Q
海外から送金する際に何日くらいで着金できますか。手数料はどのくらいですか。
A

他行での手続き次第で着金日数が違う場合があるため明確に回答ができません。当行の口座宛に海外から入金される際の手数料は不要となります。

Q
海外に居住していますが、スター不動産担保ローンは利用できますか？
A

海外に居住されているお客さまの場合、スター不動産担保ローンはご利用いただけません。

東京スターダイレクト（インターネットバンキング）

ご利用開始

Q
東京スターダイレクト(インターネットバンキング）を利用開始したいのですが、エラーが表示され、オンラインサインアップができません。
A

エラーが表示され、オンラインサインアップができない場合は、お名前、店番、口座番号、生年月日、キャッシュカード暗証番号（キャッシュカード兼電話取引用暗証番号もしくは電話取引用暗証番号）のいずれかが、当行に登録されているお客さまの情報と相違していることが原因です。表示されているエラーコードをご確認のうえ、当行担当者までご連絡をお願いいたします。

Q
東京スターダイレクト(インターネットバンキング）を利用開始したいのですが、オンラインサインアップ手続きで同意欄のチェックができません。
A

オンラインサインアップ－規約同意画面で「東京スターダイレクトに係る各種規定」、「個人情報のお取り扱いについて（プライバシーステートメント）」の各リンクをクリックし、内容をご確認いただくと、同意欄にチェックを入れらるようになります。
リンクをクリックすると別ウィンドウが立ち上がりますので、内容確認後にもとの画面へ戻りチェックボックスにチェックを入れてください。

Q
ログイン合言葉は日本語以外も登録することができますか。
A

はい、日本語以外でも登録できます。半角英数字を登録することができます。ただし、中国語文字の入力は不可となります。
非居住者口座をご利用のお客さまは、日本語カナの入力欄にアルファベット（例：ABC、abc）でご入力をおねがいいたします。

Q
東京スターダイレクト（インターネットバンキング）が利用できるのかを確認する方法を教えてください。
A

お客さまのご登録内容を確認いたしますので、当行担当者までご連絡をお願いいたしします。

Q
キャッシュカードは必ず必要ですか？キャッシュカードの発行にはどのくらい時間がかかりますか。
A

キャッシュカードの発行を希望されないお客さまでも、東京スターダイレクト（インターネットバンキング）をご利用いただけます。キャッシュカードの発行をご希望の場合は、当行の担当者までお問い合わせください。通常約2週間で、4桁の暗証番号通知書・キャッシュカードがお手元に届きます。

ログインID,パスワード,合言葉を忘れてしまった場合や、ロックがかかってしまった場合は初期化のお手続きをいたします。当行担当者までご連絡をお願いいたします。
その後、再度オンラインサインアップ（初期登録）をお願いします。

ATMサービスセンター

操作方法ほか

Q
東京スターダイレクト(インターネットバンキング）で使用しているメールアドレスを変更したい場合はどうしたらよいでしょうか。
A

メールアドレスの変更をご希望の場合は、初期化のお手続きをお願いします。お手数ですが、当行担当者までご連絡をお願いいたします。

Q
東京スターダイレクト(インターネットバンキング）で住所や電話番号の変更はできますか。
A

東京スターダイレクト（インターネットバンキング）上では住所、お電話番号の変更はできません。当行担当者までご連絡をお願いいたします。

Q
東京スターダイレクト(インターネットバンキング）では、口座開設日以降の入出金明細すべてが照会できるのでしょうか。
A

利用開始登録完了日の前月分よりご照会いただけます。

Q
残高照会と入出金明細以外のメニューがエラーになり利用することができません。
A

現在、非居住者口座（個人向け）をご利用のお客さまにご提供するサービスは、残高照会と入出金明細のみとなっております。今後、順次サービス拡大を予定しております。

Q
東京スターダイレクト（インターネットバンキング）で普通預金の過去の預金残高や取引明細を確認することはできますか。
A

残高・明細照会の「入出金明細照会」より、円普通預金の明細をご照会いただけます。
既に口座をお持ちの場合、オンラインサインアップ手続きされた日の前月1日より、ご照会いただけます。

  1. 1照会開始年月日の翌年同日まで選択可能です。（例：2025年12月1日から2026年12月1日まで）。
  2. 2照会件数が200件を超えた場合は、照会期間を変更して再度検索してください。
Q
現在契約している円定期預金や外貨預金などの明細も東京スターダイレクト（インターネットバンキング）で確認することができますか。
A

残高照会サービスでは円普通預金、外貨定期預金、円定期預金の残高をご確認いただけます。ただし、外貨普通預金、仕組み預金はサービス対象外です。

Q
東京スターダイレクト(インターネットバンキング）から口座解約手続きはできますか。
A

東京スターダイレクト（インターネットバンキング）上では口座解約手続きはできません。
口座解約方法については当行担当者までご連絡をお願いいたします。

Q
東京スターダイレクト(インターネットバンキング）のスマートフォン版とPC版で違いはありますか。
A

Q
東京スターダイレクト（インターネットバンキング）のページが翻訳されないのですがどうすればいいですか。
A

お使いのブラウザに以前のサイト情報が残っている（キャッシュが残っている）可能性がございますので、一度ブラウザのキャッシュをクリアしていただきますようお願い申し上げます。

【ブラウザのキャッシュクリア方法】

  • PCをお使いの方
    Windows: Ctrl + Shift + R を同時に押す
    Mac: Command + Shift + R を同時に押す
    上記で改善しない場合は、ブラウザの設定メニューから「閲覧履歴の削除」や「キャッシュの消去」を行ってください。
  • スマートフォンをお使いの方
    iPhone/iPad (Safari): 設定 ＞ アプリ ＞ Safari ＞ 履歴とWebサイトデータを消去
    Android (Chrome): Chromeアプリ ＞ 右上のメニュー（︙） ＞ 設定 ＞ プライバシーとセキュリティ ＞ 閲覧履歴データの削除 ＞「キャッシュされた画像とファイル」にチェックを入れて「データを削除」を行ってください。

また、Googleなど特定サービスのドメインを遮断するネットワーク設定になっている場合は、翻訳が正常に行われない場合がございますので、お使いの環境のネットワーク設定も併せてご確認ください。
